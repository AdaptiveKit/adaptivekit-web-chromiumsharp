ChromiumSharp

With ChromiumSharp you can easily leverage the power of HTML5 from within your .NET/Mono applications. Lovingly built on solid foundations so you can focus on your next masterpiece and not the tricky bits; because we ♥ what we do.

ChromiumSharp is a sub-project of AdaptiveKit – check us out.

Le Project

The aim of the project is to provide a stable yet cutting-edge version of the Chromium web browser engine that is embeddable into .NET/Mono applications and compatible with the latest Windows, Mac OSX and Linux operating systems. The project contains everything you need to start working on your next cool application without having to spend time with all the fiddly bits particular to each of the supported operating systems.

The code is sourced from carefully selected revisions for each of the ChromiumSharp project dependencies. Those projects are chain-compiled on each of the supported operating systems to ensure the maximum consistency in the resulting binaries across platforms.

ChromiumSharp is a sub-project of AdaptiveKit and is at the heart of many other related sub-projects and, as such, we take special care to make sure everything is as smooth as possible. We eat our own dogfood... so we want a rock solid build.

We hope you enjoy the work.

About Chromium

Chromium is an open-source browser project that aims to build a safer, faster, and more stable way for all users to experience the web. It’s a colossal collection of projects (that includes WebKit) that make up the foundation on which the ChromiumSharp project is built. If you wish to “roll your own” Chromium check out their site – be warned, arm yourself with plenty patience and Diet Coke… it’s going to be a long build!

The Google-authored portion of Chromium is released under the BSD license, with other parts being subject to a variety of different permissive open-source licenses, including the MIT License, the LGPL, the Ms-PL and an MPL/GPL/LGPL tri-license.

About Chromium Embedded Framework (CEF)

Chromium Embedded Framework (CEF) is an open source project founded in 2008 to develop a web browser control based on the Google Chromium project. CEF currently supports a range of programming languages and operating systems and can be easily integrated into both new and existing applications while taking advantage of the same performance and HTML5 technologies available in the Google Chrome web browser. Kudos and much ♥ to Marshall Greenblatt for this excellent piece of work.

CEF is released under the new BSD license; you can read specifics here.
